# Midi
Musical Instrument Digital Interface.
Sounds are synthetic.

Series of events with some timing information. 

Each MIDI event has information for a "channel".
Maximum 16.

Tracks are events ocurringing in parallel. Whose events can be mapped ot other channels.

Messages are 7 bits.
If MSB we got varying length encoding.

The last IDs are saved.  Based on the MSB being set you can  set all subsequent ID with the same ID.

NoteOn with velocity of 0 = NoteOff.

UTF8
